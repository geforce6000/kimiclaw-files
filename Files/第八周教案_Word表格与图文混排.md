## 第一课时：Word表格制作（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
Word表格基本操作：

1. 插入表格
   - 插入 → 表格 → 选择行列数
   - 快捷方式：直接拖动选择

2. 表格结构
   - 行：横向（水平方向）
   - 列：纵向（垂直方向）
   - 单元格：行和列交叉的小格子

3. 基本操作
   - 选中：点击单元格/拖动选择多个
   - 输入：在单元格内直接打字
   - 合并单元格：选中 → 右键 → 合并单元格
   - 拆分单元格：选中 → 右键 → 拆分单元格

4. 表格格式
   - 边框：设置表格线的样式
   - 底纹：设置单元格背景色
   - 对齐：单元格内文字对齐方式
```

**表格应用场景**：
- 课程表、成绩单
- 信息登记表
- 数据统计表

---

### 二、考点强化记忆（20分钟）

**考点1：表格结构（填空题）**

教师带领学生记忆：
```
表格由______和______组成。
横向叫______，纵向叫______。
答案：行、列；行、列
```

**考点2：合并单元格（选择题）**

教师强调：
- 合并：多个单元格变一个
- 拆分：一个单元格变多个

**考点3：表格对齐（判断题）**

教师讲解：
- 表格可以居中对齐
- 单元格内文字可以左/中/右对齐

---

### 三、例题讲解（8分钟）

**例题1：填空题**
在Word中，表格由横向的______和纵向的______组成。

**讲解要点**：
- 答案：行、列
- 记忆：横着走叫行，竖着走叫列

**例题2：选择题**
要将表格中的多个单元格合并成一个，应该使用（ ）
A. 拆分单元格  B. 合并单元格  C. 删除单元格  D. 插入单元格

**讲解要点**：
- 合并单元格：把多个合成一个
- 答案：B

**例题3：判断题**
在表格单元格内，只能输入文字，不能插入图片。（ ）

**讲解要点**：
- 单元格内可以输入文字、插入图片、设置格式
- 答案：×

### 四、课堂小结与记忆任务（2分钟）

**本课必背内容**：
1. 表格结构：行（横）、列（纵）
2. 基本操作：插入表格、合并单元格
3. 应用场景：课程表、成绩单

**课后记忆任务**：

## 第二课时：图文混排基础（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
Word图文混排：

1. 插入图片
   - 插入 → 图片 → 此设备
   - 选择图片文件 → 插入

2. 图片格式设置
   - 大小：拖动角点调整
   - 环绕方式：
     * 嵌入型（默认，图片当文字处理）
     * 四周型（文字环绕图片四周）
     * 浮于文字上方（图片盖在文字上）

3. 插入形状
   - 插入 → 形状 → 选择形状
   - 常用：矩形、圆形、箭头、线条

4. 插入文本框
   - 插入 → 文本框
   - 可在任意位置放置文字

图文混排原则：
- 图片大小适中
- 图文搭配协调
- 整体美观整洁
```

---

### 二、考点强化记忆（20分钟）

**考点1：图片环绕方式（选择题）**

教师讲解三种常用方式：
- 嵌入型：图片像文字一样
- 四周型：文字环绕图片
- 浮于文字上方：图片在最上面

**考点2：插入操作（填空题）**

教师带领学生记忆：
- 插入图片：插入 → 图片
- 插入形状：插入 → 形状

**考点3：文本框（判断题）**

教师强调：
- 文本框可以放在页面任意位置
- 用于在特殊位置放置文字

---

### 三、例题讲解（8分钟）

**例题1：选择题**
要让文字环绕在图片四周，应设置的环绕方式是（ ）
A. 嵌入型  B. 四周型  C. 浮于文字上方  D. 衬于文字下方

**讲解要点**：
- 四周型：文字在图片四周环绕
- 答案：B

**例题2：填空题**
在Word中，插入图片的菜单是______选项卡。

**讲解要点**：
- 所有插入操作都在"插入"选项卡
- 答案：插入

**例题3：判断题**
文本框只能在页面中间位置插入，不能移动到其他地方。（ ）

**讲解要点**：
- 文本框可以拖动到任意位置
- 答案：×

---

### 四、课堂小结与记忆任务（2分钟）

**本课必背内容**：
1. 插入图片：插入 → 图片
2. 环绕方式：嵌入型、四周型、浮于文字上方
3. 插入形状：插入 → 形状

**课后记忆任务**：
- 记住三种环绕方式的区别
- 理解图文混排的基本概念

## 第三课时：表格与图文混排实践（上机教学）

### 一、上机任务说明（5分钟）

**今日任务目标**：
1. 创建表格并设置格式
2. 插入图片并设置环绕方式
3. 简单图文混排

**强调**：表格和图片都要适当调整大小

---

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________  机位号：________

□ 任务一：创建课程表（15分）
   操作步骤：
   ① 打开Word，新建文档
   ② 插入 → 表格 → 6列×6行
   ③ 第一行输入：节次、周一、周二、周三、周四、周五
   ④ 第一列输入：第1节、第2节、第3节、第4节、午休、第5节
   ⑤ 合并"午休"所在行的中间单元格
   ⑥ 为表格添加边框和底纹（浅色）
   ⑦ 保存文档

□ 任务二：插入图片（10分）
   操作步骤：
   ① 新建段落，输入"校园风景"
   ② 插入 → 图片 → 此设备
   ③ 选择一张图片插入
   ④ 调整图片大小（拖动角点，保持比例）
   ⑤ 设置环绕方式：四周型
   ⑥ 观察文字环绕效果

□ 任务三：插入形状和文本框（10分）
   操作步骤：
   ① 插入 → 形状 → 选择矩形
   ② 在页面上画一个矩形
   ③ 设置矩形填充颜色（浅色）
   ④ 插入 → 文本框 → 绘制文本框
   ⑤ 在文本框内输入"我的作品"
   ⑥ 将文本框拖动到矩形上方

【备用任务】完成后可做：
□ 尝试其他形状（圆形、箭头）
□ 设置不同的图片环绕方式
```

**教师巡视要点**：
- 指导表格行列数的选择
- 帮助不会合并单元格的学生
- 提醒图片大小要适中

---

### 三、任务单回收与点评（5分钟）

**回收任务单**，快速点评：
- 强调易错点：
  - 合并单元格要先选中多个单元格
  - 调整图片大小要拖动角点，保持比例
  - 环绕方式选择四周型才能看到文字环绕效果

**布置课后任务**：
- 制作一张个人信息表（姓名、年龄、爱好等）
- 期中考试复习准备
- 准备下节课默写：表格结构、图片环绕方式
