### 三、内容讲解（25分钟）

#### 1. 构图的方法（12分钟）

**概念讲解**：
```
构图 = 把画面元素安排在合适的位置
```

**三种基础构图法**：

| 构图法 | 特点 | 适用场景 |
|-------|------|---------|
| 三分法 | 画面分成九宫格，主体放交叉点 | 风景、人像 |
| 中心构图 | 主体放正中间 | 特写、对称物体 |
| 对角线构图 | 主体沿对角线排列 | 道路、河流 |

**口诀**：三分法最常用，中心构图突出主体，对角线有动感

#### 2. 色彩的原理（10分钟）

**基础概念**：
```
三原色：红、黄、蓝
冷暖色：
- 暖色（红、橙、黄）——热情、温暖
- 冷色（蓝、绿、紫）——冷静、清新
```

**美图秀秀调色功能**：
- 亮度：明暗程度
- 对比度：明暗差别
- 饱和度：颜色鲜艳程度

#### 3. 胸牌设计要素（3分钟）

**讲解胸牌组成**：
- 照片（人像）
- 姓名
- 班级/单位
- 装饰元素

### 四、简单练习（8分钟）

**练习1**：判断题
```
三分法构图是把画面分成九宫格，主体放在交叉点。（ ）
答案：√
```

**练习2**：选择题
```
想拍出温暖的感觉，应该多用（ ）色调。
A. 蓝色  B. 红色  C. 绿色
答案：B（红色是暖色）
```

**练习3**：填空题
```
美图秀秀调色的三个主要参数是：______、对比度、______。
答案：亮度、饱和度
```

## 第二课时：制作"大国小工匠"胸牌（上机教学）

### 一、上机任务说明（5分钟）

**本周任务目标**：
- 用美图秀秀制作一张胸牌
- 应用构图和色彩知识
- 添加文字和装饰

**对应关系**：
- 教室学了构图和色彩
- 上机实践：制作胸牌（图像处理应用）

---

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________ 机位号：________

□ 任务一：打开图片（3分）
   ① 打开美图秀秀
   ② 打开自己准备的证件照（或示例人像图）

□ 任务二：裁剪构图（8分）
   ① 点击"裁剪"按钮
   ② 选择"证件照"比例（1寸或2寸）
   ③ 调整裁剪框，把人像放在中间或偏上位置
   ④ 点击"完成"

□ 任务三：调色美化（8分）
   ① 点击"美化"按钮
   ② 调整亮度：让人脸明亮清晰
   ③ 调整对比度：不要太灰
   ④ 调整饱和度：肤色自然
   ⑤ 可应用一个轻度美颜

□ 任务四：添加文字信息（10分）
   ① 点击"文字"按钮
   ② 输入姓名（大号字体，放顶部）
   ③ 输入班级（中号字体，放底部）
   ④ 输入"大国小工匠"（装饰文字，放侧面）
   ⑤ 选择合适的字体颜色（建议深色）

□ 任务五：添加边框装饰（4分）
   ① 点击"边框"按钮
   ② 选择一个简洁的边框
   ③ 调整边框宽度

□ 任务六：保存作品（2分）
   ① 点击"保存"
   ② 文件名：学号_姓名_胸牌.jpg
   ③ 保存到D盘

【拓展任务】（完成快的同学）：
□ 尝试添加"贴纸"装饰
□ 尝试"背景虚化"效果
```

**教师巡视要点**：
- 指导裁剪比例选择
- 提醒调色要自然，不要过度美颜
- 检查文字信息是否完整
- 帮助不会添加边框的学生

---
