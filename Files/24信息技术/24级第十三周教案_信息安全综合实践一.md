### 三、内容讲解（25分钟）

#### 1. 常见信息安全事件（12分钟）

**网络诈骗类型**：

| 类型 | 套路 | 防范方法 |
|-----|------|---------|
| 冒充客服 | "您的快递丢了，要退款" | 官方渠道核实 |
| 虚假中奖 | "您中了iPhone，点击领取" | 不点击陌生链接 |
| 钓鱼网站 | 假冒银行/QQ登录页面 | 核对网址 |
| 杀猪盘 | 先谈恋爱再骗投资 | 不轻信网友 |
| 刷单返利 | "先垫付小额，再要求大额" | 不参与刷单 |

**口诀**：陌生链接不要点，转账汇款要核实

#### 2. 个人信息保护（8分钟）

**敏感信息**：
- 身份证号
- 银行卡号
- 手机号
- 家庭住址
- 密码

**保护措施**：
```
1. 不随意填写个人信息
2. 不晒含敏感信息的照片（如火车票、身份证）
3. 不同网站用不同密码
4. 定期修改密码
5. 开启双重验证
```

#### 3. 遇到诈骗怎么办（5分钟）

**应对步骤**：
1. 保持冷静，不要慌张
2. 核实信息（打电话确认）
3. 不转账、不汇款
4. 及时报警（110）
5. 保留证据（截图、录音）

```

```

### 二、分任务操作练习（35分钟）

```

□ 任务一：识别钓鱼网站（10分）
   操作步骤：
   ① 打开浏览器
   ② 搜索"钓鱼网站识别方法"
   ③ 学习3个识别技巧：

□ 任务二：了解账号安全设置（8分）
   操作步骤：

      - 绑定手机
      - 设置密保问题
      - 开启登录保护

   ③ 记录：账号安全需要____、____、____

□ 任务三：制作防诈骗宣传页（15分）
   操作步骤：

   ⑥ 内容页3：遇到诈骗怎么办（3个步骤）
   ⑦ 插入相关警示图片
   ⑧ 保存：学号_姓名_防诈骗宣传.pptx

□ 任务四：搜索学习（2分）
```
