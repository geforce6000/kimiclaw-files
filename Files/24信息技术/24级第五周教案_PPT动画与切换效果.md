### 三、内容讲解（25分钟）

#### 1. 动画效果（12分钟）

**概念讲解**：
```
动画 = 让幻灯片里的文字、图片动起来
```

**动画类型**：

| 动画类型 | 效果 | 用途 |
|---------|------|------|
| 进入动画 | 飞入、淡入、弹入 | 元素出现 |
| 强调动画 | 放大、变色、闪烁 | 引起注意 |
| 退出动画 | 飞出、淡出 | 元素消失 |

**添加步骤**：
1. 选中要加动画的元素
2. 点击"动画"选项卡
3. 选择动画效果
4. 设置触发方式（点击/自动）

**注意**：动画不要太多，一页2-3个即可

#### 2. 切换效果（10分钟）

**概念讲解**：
```
切换 = 幻灯片之间的过渡效果
```

**常用切换效果**：
- 淡入淡出（最常用，简洁）
- 推进（像翻书）
- 擦除（像擦黑板）
- 覆盖（新页覆盖旧页）

**设置步骤**：
1. 点击"切换"选项卡
2. 选择切换效果
3. 应用到当前页或全部

**注意**：一个PPT最好用同一种切换效果

#### 3. 放映设置（3分钟）

**放映方式**：
- 从头开始：F5键
- 从当前页：Shift+F5
- 自动放映：设置排练时间

### 四、简单练习（8分钟）

**练习1**：填空题
```
让幻灯片里的文字动起来，应该设置______效果。
答案：动画
```

**练习2**：选择题
```
幻灯片之间的过渡效果叫（ ）。
A. 动画  B. 切换  C. 主题  D. 版式
答案：B
```

**练习3**：判断题
```
PPT的动画越多越好，每一页都要加很多动画。（ ）
答案：×（动画要适度，太多会分散注意力）
```

### 五、课堂小结与作业（2分钟）

**本课重点**：
1. 动画：让元素动起来（进入、强调、退出）
2. 切换：页面过渡效果（淡入淡出最常用）
3. 放映：F5从头播放

**课后作业**：

## 第二课时：为PPT添加动画（上机教学）

### 一、上机任务说明（5分钟）

**本周任务目标**：

**对应关系**：

---

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________ 机位号：________

□ 任务一：打开上节课PPT（3分）
   ① 打开PowerPoint
   ② 打开"学号_姓名_家乡.pptx"

□ 任务二：添加标题动画（8分）
   ① 选中标题文字"我的家乡"
   ② 点击"动画"选项卡
   ③ 选择"飞入"效果
   ④ 选择"自顶部"方向
   ⑤ 预览效果（点击"预览"按钮）

□ 任务三：添加图片动画（8分）
   ① 选中图片
   ② 点击"动画"
   ③ 选择"淡入"效果
   ④ 设置"上一动画之后"自动播放

□ 任务四：设置切换效果（8分）
   ① 点击"切换"选项卡
   ② 选择"淡入"效果
   ③ 点击"应用到全部"
   ④ 设置切换时间：2秒

□ 任务五：添加新动画页（5分）
   ① 新建一张幻灯片
   ② 标题：谢谢观看
   ③ 给标题添加"弹跳"动画
   ④ 这页作为结束页

□ 任务六：放映测试（2分）
   ① 按F5从头放映
   ② 点击鼠标看动画效果
   ③ 检查切换是否流畅
   ④ 按Esc退出放映

□ 任务七：保存（1分）
   ① 保存文件（Ctrl+S）

【拓展任务】（完成快的同学）：
□ 给文字添加"强调"动画（变色或放大）
□ 尝试其他切换效果（推进、擦除）
```

**教师巡视要点**：
- 帮助找到上节课文件的学生
- 指导选择动画效果
- 提醒设置"自动播放"或"点击播放"
- 指导设置切换时间

## 附件：本周知识点

### 理论知识点
1. **动画类型**：进入、强调、退出
2. **切换效果**：淡入淡出、推进、擦除
3. **放映快捷键**：F5（从头）、Shift+F5（当前页）
4. **原则**：动画适度，切换统一

### 上机操作要点
1. 选中元素→动画选项卡→选择效果
2. 切换选项卡→选择效果→应用到全部
3. 设置自动播放或点击播放
4. F5放映测试效果
5. Ctrl+S保存
