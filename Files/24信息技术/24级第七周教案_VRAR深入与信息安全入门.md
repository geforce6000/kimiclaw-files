### 三、内容讲解（25分钟）

#### 1. VR/AR应用领域（12分钟）

**VR应用**：
```
1. VR游戏：打僵尸、过山车、滑雪
2. VR教育：虚拟实验室、太空探索
3. VR旅游：足不出户游世界
4. VR培训：飞行员、医生模拟训练
5. VR看房：不用去现场看房子
```

**AR应用**：
```
1. 支付宝扫福字
2. 抖音特效（兔耳朵、变装）
3. AR导航（路面上显示箭头）
4. AR购物（试衣服、试鞋子）
5. 游戏Pokemon Go（抓精灵）
```

**记忆口诀**：VR进虚拟玩游戏，AR看现实加信息

#### 2. 什么是信息安全（8分钟）

**概念讲解**：
```
信息安全 = 保护信息不被泄露、篡改、破坏
```

**信息安全五要素**：

| 要素 | 含义 | 例子 |
|-----|------|------|
| 保密性 | 不被别人看到 | 密码不告诉他人 |
| 完整性 | 不被篡改 | 文件不被修改 |
| 可用性 | 随时能用 | 系统不宕机 |
| 可控性 | 能控制访问 | 设置权限 |
| 不可否认性 | 行为可追溯 | 操作有记录 |

**口诀**：保密完整可用，可控不可否认

#### 3. 常见信息安全风险（5分钟）

**风险类型**：
- 病毒木马：偷密码、偷文件
- 网络诈骗：假冒网站、钓鱼链接
- 信息泄露：个人信息被卖
- 账号被盗：密码太简单

### 四、简单练习（8分钟）


### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________ 机位号：________

□ 任务一：体验VR视频（10分）
   操作步骤：
   ① 打开浏览器
   ② 搜索"VR故宫全景"
   ③ 找一个可以在线观看的VR视频
   ④ 用鼠标拖动画面，体验360度视角
   ⑤ 感受：仿佛真的在故宫里

□ 任务二：设置强密码（10分）
   操作步骤：
   ① 在Word中输入密码设置原则：
      - 至少8位
      - 包含大小写字母
      - 包含数字
      - 包含特殊符号（如@#$）
   ② 设计一个自己的强密码示例：
      例如：Jxkj2024@ 或 MyPhone#8
   ③ 保存到文档中

□ 任务三：制作安全提示卡（10分）
   操作步骤：
   ① 打开PowerPoint
   ② 新建演示文稿
   ③ 标题：信息安全小贴士
   ④ 内容页1：设置强密码（3条要点）
   ⑤ 内容页2：不点陌生链接（提醒）
   ⑥ 内容页3：保护个人信息（建议）
   ⑦ 保存：学号_姓名_安全提示.pptx

□ 任务四：了解杀毒软件（5分）
   操作步骤：
   ① 搜索"国产杀毒软件"
   ② 了解3个国产杀毒软件名称
   ③ 记录：____、____、____

【拓展任务】（完成快的同学）：
□ 搜索"钓鱼网站识别方法"
□ 了解什么是"电信诈骗"
```
