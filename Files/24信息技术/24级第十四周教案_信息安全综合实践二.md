### 三、内容讲解（25分钟）

#### 1. 防火墙（10分钟）

**概念讲解**：
```
防火墙 = 网络的"门卫"
功能：
- 拦住可疑访问
- 防止黑客入侵
- 控制网络进出
```

**防火墙类型**：
- 软件防火墙：安装在电脑上（如Windows防火墙）
- 硬件防火墙：企业用的专用设备

**个人电脑防护**：
- 开启Windows防火墙
- 不关闭安全软件
- 不随意允许程序联网

#### 2. 杀毒软件（8分钟）

**功能**：
- 查杀病毒木马
- 实时防护
- 定期扫描

**国产杀毒软件**：
- 360安全卫士
- 火绒安全
- 腾讯电脑管家
- 金山毒霸

**使用建议**：
- 安装一款即可，不要多装
- 定期更新病毒库
- 定期全盘扫描

#### 3. 国产操作系统（7分钟）

**为什么要用国产系统**：
```
- 安全可控：不被国外卡脖子
- 保护隐私：数据不被窃取
- 支持国产：发展自己的技术
```

**国产操作系统**：
- 银河麒麟（Linux基础）
- 统信UOS
- 鸿蒙系统（手机/平板）

**现状**：
- 越来越好用
- 兼容更多软件
- 政府、企业逐步推广

```

**练习3**：判断题

```

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________ 机位号：________

□ 任务一：规划手册结构（5分）
   手册主题：个人信息安全手册
   结构：
   - 封面：标题+姓名
   - 第一章：密码安全
   - 第二章：防范诈骗
   - 第三章：个人信息保护
   - 第四章：电脑安全防护
   - 结语：安全口诀

□ 任务二：制作封面和第一章（10分）
   第一章-密码安全：
   - 什么是强密码
   - 密码设置原则
   - 不同账号不同密码
   - 定期修改密码

□ 任务三：制作第二、三章（15分）
   
   第二章-防范诈骗：
   - 常见诈骗类型（3种）
   - 识别方法
   - 遇到怎么办
   
   第三章-个人信息保护：
   - 什么是敏感信息
   - 朋友圈不晒什么
   - 如何保护隐私

□ 任务四：制作第四章和结语（3分）
   第四章-电脑安全防护：
   - 安装杀毒软件
   - 开启防火墙
   - 定期更新系统
   
   结语-安全口诀：
   如：陌生链接不点击，陌生电话不轻信

□ 任务五：排版保存（2分）
   ① 设置标题格式（加粗、字号）
   ② 正文分点清晰
   ③ 文件名：学号_姓名_信息安全手册.docx
   ④ 保存到D盘

【展示环节】

```
