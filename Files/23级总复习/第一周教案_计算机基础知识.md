## 第一课时：信息技术基础（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
信息技术 = 获取、处理、传递、利用信息的技术

四大关键词：获取 → 处理 → 传递 → 利用

注意：信息技术 ≠ 计算机技术
      （计算机只是实现信息技术的工具之一）
```

**典型应用回顾**：
- 通信类：微信、QQ、电话
- 购物类：淘宝、拼多多、美团
- 出行类：高德地图、滴滴打车
- 学习类：百度、抖音学习视频

**信息社会三特征**（口诀）：
```
数字化（纸质→电子）
网络化（不出门办万事）
智能化（手机懂你喜好）
```

---

### 二、考点强化记忆（20分钟）

**考点1：信息技术定义（必考填空）**

教师带领学生齐读3遍，然后默写：
```
信息技术是指信息的______、______、______和______的技术。
答案：获取、处理、传递、利用
```

**记忆技巧讲解**：
- "获取"：像拍照、录音
- "处理"：像美颜、剪辑
- "传递"：像发微信、打电话
- "利用"：像导航、查天气

**考点2：信息技术应用举例**

教师提问，学生集体回答，巩固记忆。

**考点3：信息社会特征（选择题常见）**

教师举例说明：
- "用现金买东西"——不属于信息社会特征（不是数字化）
- "网上购物"——属于信息社会特征（网络化）

### 三、例题讲解（8分钟）

**例题1：填空题**
信息技术的核心是信息的______、______、______和______。

**讲解要点**：
- 强调四个关键词的顺序可以调整，但四个词必须完整
- 常见错误：漏写"利用"或写成"使用"

**例题2：填空题**
列举两个信息技术在生活中的应用：______、______。

**讲解要点**：
- 只要是利用信息技术的例子都可以
- 参考答案：微信聊天、网上购物、手机支付、百度查资料等

**例题3：选择题**
下列不属于信息技术应用的是（ ）
A. 用微信聊天  B. 用淘宝购物  C. 用现金买菜  D. 用百度查资料

**讲解要点**：
- 分析每个选项：A、B、D都用到了信息技术
- C选项用现金，没有使用信息技术手段
- 答案：C

## 第二课时：计算机硬件基础（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
计算机五大组成部分：
┌─────────┐    ┌─────────┐    ┌─────────┐
│ 输入设备 │ → │  主机   │ → │ 输出设备 │
│(键盘鼠标)│    │(CPU+内存 │    │(显示器) │
└─────────┘    │+硬盘)   │    └─────────┘
               └─────────┘
                    ↑
               ┌─────────┐
               │ 存储设备 │
               │(硬盘/U盘)│
               └─────────┘
```

**硬件识别速记表**：

| 设备名称 | 功能 | 归类 | 考点关键词 |
|---------|------|------|-----------|
| 键盘、鼠标 | 向电脑输入指令 | 输入设备 | 人→电脑 |
| 显示器、打印机 | 显示/打印结果 | 输出设备 | 电脑→人 |
| 硬盘、U盘 | 长期存储数据 | 外存储器 | 断电数据在 |
| 内存条 | 临时存储数据 | 内存储器 | 断电数据丢 |
| CPU | 处理数据 | 中央处理器 | 计算机大脑 |

**考点强调**：
- CPU = 中央处理器（全称必背）
- 内存 vs 硬盘区别：
  - 内存：速度快、容量小、断电丢数据
  - 硬盘：速度慢、容量大、断电存数据

---

### 二、考点强化记忆（20分钟）

**考点1：硬件分类（选择题高频）**

教师快速提问，学生集体回答：
- 键盘 → 输入设备
- 显示器 → 输出设备
- 硬盘 → 存储设备
- CPU → 运算+控制

**考点2：存储单位换算（必考）**

板书（从小到大）：
```
B（字节）→ KB → MB → GB

1 KB = 1024 B
1 MB = 1024 KB
1 GB = 1024 MB

记忆：一个汉字 = 2B
      一首歌 ≈ 5MB
      一部电影 ≈ 1GB
```

**换算练习讲解**：
1. 2048 B = ______ KB （答案：2）
   - 讲解：2048 ÷ 1024 = 2

2. 1024 MB = ______ GB （答案：1）
   - 讲解：1024 MB = 1 GB

**考点3：CPU（填空题必考）**

默写：CPU的中文名称是______。（中央处理器）

---

### 三、例题讲解（8分钟）

**例题1：选择题**
下列属于输入设备的是（ ）
A. 显示器  B. 打印机  C. 鼠标  D. 音箱

**讲解要点**：
- 分析：显示器、打印机、音箱都是电脑向外输出信息的设备
- 鼠标是向电脑输入指令的设备
- 答案：C

**例题2：选择题**
计算机中，断电后数据会丢失的是（ ）
A. 硬盘  B. 内存  C. U盘  D. 光盘

**讲解要点**：
- 分析：硬盘、U盘、光盘都是外存，断电后数据还在
- 内存是临时存储，断电后数据丢失
- 答案：B

**例题3：选择题**
CPU的中文名称是（ ）
A. 中央处理器  B. 内存  C. 硬盘  D. 显卡

**讲解要点**：
- 这是必背知识点，CPU = 中央处理器
- 答案：A

**例题4：填空题**
计算机存储容量的基本单位是______。

**讲解要点**：
- 基本单位是字节，英文是 Byte，简写 B
- 答案：字节（或 B）

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________  机位号：________

□ 任务一：规范开机（5分）
   操作步骤：
   ① 先打开显示器电源
   ② 再按主机电源键
   ③ 等待进入Windows桌面
   
   检查点：能否正常进入桌面？ □是 □否

□ 任务二：计算器数制转换（15分）
   操作步骤：
   ① 开始菜单 → Windows附件 → 计算器
   ② 点击左上角"≡" → 选择"程序员"
   ③ 选择"DEC"（十进制），输入数字
   ④ 查看"BIN"对应的二进制数
   
   练习题（把答案填在括号里）：
   十进制 5  = 二进制 (      )
   十进制 8  = 二进制 (      )
   十进制 10 = 二进制 (      )
   十进制 15 = 二进制 (      )
   
   提示：可以用计算器验证

□ 任务三：查看电脑配置（10分）
   操作步骤：
   ① 在桌面"此电脑"图标上点右键
   ② 选择"属性"
   ③ 找到以下信息并填写：
      
      处理器(CPU)：________________
      内存大小：________________
      系统类型：________________

□ 任务四：规范关机（5分）
   操作步骤：
   ① 开始菜单 → 电源 → 关机
   ② 等待主机电源灯熄灭
   ③ 关闭显示器电源
   
   检查点：是否正确关机？ □是 □否

【备用任务】完成后可做：
□ 打开记事本，输入"姓名+学号"，保存到桌面
```

**教师巡视要点**：
- 观察开关机顺序是否正确（先显示后主，关机相反）
- 帮助不会用计算器的学生
- 提醒：不要直接按电源键关机
