## 第一课时：操作系统基础（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
操作系统 = 管理计算机硬件与软件资源的程序

三大功能：
1. 管理硬件（CPU、内存、硬盘等）
2. 管理软件（程序的安装、运行、卸载）
3. 提供用户界面（图形界面、命令行界面）

主流操作系统：
- Windows（最常用）
- 国产：银河麒麟、Deepin（统信UOS）、鸿蒙
- 手机：Android、iOS
```

**考点速记**：
- 操作系统是计算机的"大管家"
- 没有操作系统，计算机无法使用
- 国产操作系统名称：银河麒麟、Deepin、鸿蒙

---

### 二、考点强化记忆（20分钟）

**考点1：操作系统的功能（填空题）**

教师带领学生齐读3遍，然后默写：
```
操作系统的主要功能：
1. ______管理（硬件资源）
2. ______管理（软件资源）
3. 提供______界面（用户操作界面）

答案：1. 硬件  2. 软件  3. 用户
```

**考点2：主流操作系统识别（选择题）**

教师讲解分类：
- 电脑用：Windows、银河麒麟、Deepin
- 手机用：Android、iOS、鸿蒙

**考点3：国产操作系统（必考）**

必背三个名称：
1. 银河麒麟
2. Deepin（或统信UOS）
3. 鸿蒙

---

### 三、例题讲解（8分钟）

**例题1：填空题**
操作系统是管理计算机______和______资源的系统软件。

**讲解要点**：
- 答案：硬件、软件
- 强调：操作系统本身也是软件，是最基础的软件

**例题2：选择题**
下列属于国产操作系统的是（ ）
A. Windows  B. Android  C. 银河麒麟  D. iOS

**讲解要点**：
- Windows是美国微软的
- Android是谷歌的（美国）
- iOS是苹果的（美国）
- 银河麒麟是中国的
- 答案：C

**例题3：选择题**
操作系统的功能不包括（ ）
A. 管理CPU  B. 管理内存  C. 编辑文档  D. 管理文件

**讲解要点**：
- A、B、D都是操作系统的管理功能
- C选项"编辑文档"是应用软件（如Word）的功能
- 答案：C

## 第二课时：计算机配置与设备（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
计算机主要性能指标：

1. CPU（中央处理器）
   - 主频：GHz（如2.5GHz）
   - 核心数：双核、四核、八核
   - 口诀：主频越高越快，核心越多越强

2. 内存（RAM）
   - 容量：GB（如8GB、16GB）
   - 口诀：越大越好，至少8GB

3. 硬盘（存储）
   - 容量：GB/TB（如512GB、1TB）
   - 类型：机械硬盘（HDD）、固态硬盘（SSD）
   - 口诀：SSD比HDD快

4. 显示器
   - 分辨率：如1920×1080
   - 口诀：数字越大越清晰
```

**设备连接接口速记**：
- USB接口：接鼠标、键盘、U盘
- HDMI接口：接显示器、投影仪
- 网线接口：接网线
- 音频接口：接耳机、音箱

---

### 二、考点强化记忆（20分钟）

**考点1：性能指标识别（选择题）**

教师带领学生识别：
- CPU看主频（GHz）和核心数
- 内存看容量（GB）
- 硬盘看容量（GB/TB）和类型（SSD>HDD）

**考点2：存储单位换算（复习）**

复习上周内容：
- 1 GB = 1024 MB
- 1 TB = 1024 GB
- 一个汉字 = 2B

**考点3：设备接口识别（识图题）**

教师画图或描述：
- USB：扁长方形，最常见
- HDMI：梯形，接显示器
- 网线：比电话线宽，有卡扣

---

### 三、例题讲解（8分钟）

**例题1：选择题**
衡量CPU性能的主要指标是（ ）
A. 重量  B. 主频和核心数  C. 颜色  D. 品牌

**讲解要点**：
- CPU性能看主频（GHz）和核心数（几核）
- 主频越高、核心数越多，性能越好
- 答案：B

**例题2：填空题**
计算机内存容量为8GB，硬盘容量为512GB。其中，断电后数据会丢失的是______。

**讲解要点**：
- 内存（RAM）断电数据丢失
- 硬盘断电数据保留
- 答案：内存（或RAM、8GB内存）

**例题3：选择题**
下列接口中，用于连接U盘的是（ ）
A. HDMI接口  B. USB接口  C. 网线接口  D. 电源接口

**讲解要点**：
- U盘使用USB接口
- HDMI接显示器，网线接口接网线
- 答案：B

---

### 四、课堂小结与记忆任务（2分钟）

4. USB接口：接U盘、鼠标、键盘

**课后记忆任务**：

## 第三课时：操作系统基本操作（上机教学）

### 一、上机任务说明（5分钟）

**今日任务目标**：
1. 查看系统信息（操作系统版本、CPU、内存）
2. 创建文件夹和文本文件
3. 设置桌面背景

**强调**：操作步骤严格按照任务单执行

---

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________  机位号：________

□ 任务一：查看系统信息（10分）
   操作步骤：
   ① 右键点击"此电脑" → 选择"属性"
   ② 查看并填写以下信息：
      
      操作系统：________________
      处理器(CPU)：________________
      内存大小：________________
      系统类型：________________（32位/64位）

□ 任务二：创建文件夹和文件（15分）
   操作步骤：
   ① 在桌面上右键 → 新建 → 文件夹
   ② 文件夹命名为"第3周作业"
   ③ 打开文件夹 → 右键 → 新建 → 文本文档
   ④ 命名为"学号+姓名.txt"
   ⑤ 打开文本文件，输入：
      我的CPU是______，内存是______。
   ⑥ 保存文件（Ctrl+S）

□ 任务三：设置桌面背景（10分）
   操作步骤：
   ① 在桌面空白处右键 → 个性化
   ② 选择"背景"
   ③ 选择一张系统自带的图片
   ④ 关闭设置窗口

【备用任务】完成后可做：
□ 调整屏幕分辨率（右键桌面 → 显示设置 → 分辨率）
```

**教师巡视要点**：
- 帮助找不到"此电脑"图标的学生
- 提醒保存文件（Ctrl+S）
- 观察学生是否能独立完成文件夹创建

---

### 三、任务单回收与点评（5分钟）

**回收任务单**，快速点评：
- 强调易错点：
  - 文件夹和文件命名要规范（不要用特殊符号）
  - 文本文件编辑后要保存
  - 查看系统信息的路径：右键"此电脑"→属性

**布置课后任务**：
- 回家查看家里电脑的配置（CPU、内存、操作系统）
- 准备下节课默写：国产操作系统名称、CPU性能指标
