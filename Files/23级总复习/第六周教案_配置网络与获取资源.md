## 第一课时：网络设备与配置（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
常见网络设备：

1. 路由器（Router）
   - 功能：连接不同网络，分配IP地址
   - 特点：家庭/办公上网的核心设备
   - 标识：有天线，有多个网线接口

2. 交换机（Switch）
   - 功能：连接多台电脑，实现局域网通信
   - 特点：只有网线接口，无天线

3. 调制解调器（Modem，俗称"猫"）
   - 功能：将电话线/光纤信号转换为网络信号
   - 现代多集成在路由器中

家庭上网必备：
- 宽带账号（运营商提供）
- 路由器
- 网线或WiFi连接
```

**网络故障排除口诀**：
- 第一步：看指示灯（电源灯、网络灯）
- 第二步：重启设备（拔电源再插上）
- 第三步：检查网线（是否插紧）
- 第四步：联系运营商

### 二、考点强化记忆（20分钟）

**考点1：网络设备识别（选择题）**

教师带领学生识别：
- 有天线+多个网口 → 路由器
- 只有网口无天线 → 交换机

**考点2：路由器功能（填空题）**

教师带领学生记忆：
```
路由器的两大功能：
1. ______不同网络（连接/路由）
2. ______IP地址（分配）
```

**考点3：简单故障排除（简答题）**

教师讲解四步法：
1. 看指示灯
2. 重启设备
3. 检查网线
4. 联系运营商

### 三、例题讲解（8分钟）

**例题1：选择题**
家庭上网时，用于连接宽带并分配IP地址的设备是（ ）
A. 显示器  B. 键盘  C. 路由器  D. 打印机

**讲解要点**：
- 路由器负责连接宽带并给设备分配IP地址
- 答案：C

**例题2：填空题**
当网络出现故障时，最简单的排除方法是______设备。

**讲解要点**：
- 答案：重启（或重新启动）
- 解释：重启可以解决大部分临时故障

**例题3：选择题**
下列设备中，用于连接多台电脑实现局域网通信的是（ ）
A. 路由器  B. 交换机  C. 显示器  D. 打印机

**讲解要点**：
- 交换机专门用于局域网内多台设备连接
- 路由器侧重连接不同网络（如家庭内网与互联网）
- 答案：B

### 四、课堂小结与记忆任务（2分钟）

**本课必背内容**：
1. 路由器功能：连接网络、分配IP地址
2. 交换机功能：连接多台电脑，局域网通信
3. 故障排除四步法：看灯→重启→查线→联系

**课后记忆任务**：

- 记忆：路由器有两个功能（连接网络、分配IP）

## 第二课时：网络资源检索（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
搜索引擎：帮助我们在互联网上查找信息的工具

常用搜索引擎：
- 百度（www.baidu.com）- 最常用
- 必应（www.bing.com）- 微软出品
- 搜狗（www.sogou.com）

搜索技巧：
1. 关键词准确（不要输入整句话）
2. 使用引号精确匹配（如"信息技术"）
3. 使用减号排除（如 苹果 -手机）

网络资源类型：
- 文本：文章、论文、新闻
- 图片：JPG、PNG格式
- 音频：MP3格式
- 视频：MP4格式
```

**信息评估要点**：
- 看来源：官方网站>个人博客
- 看时间：最新信息更可靠
- 看作者：专业机构>匿名发布

---

### 二、考点强化记忆（20分钟）

**考点1：搜索引擎（填空题）**

教师带领学生记忆：
- 国内最常用：百度
- 网址：www.baidu.com

**考点2：搜索技巧（选择题）**

教师讲解：
- 关键词要精简准确
- 多个关键词用空格隔开

**考点3：信息可信度（判断题）**

教师强调：
- 官方网站信息更可靠
- 最新信息更准确
- 注意辨别虚假信息

---

### 三、例题讲解（8分钟）

**例题1：填空题**
国内最常用的中文搜索引擎是______。

**讲解要点**：
- 答案：百度

**例题2：选择题**
在搜索引擎中，使用双引号""的作用是（ ）
A. 排除关键词  B. 精确匹配短语  C. 搜索图片  D. 限制时间

**讲解要点**：
- 双引号表示精确匹配，搜索结果必须包含引号内的完整短语
- 答案：B

**例题3：判断题**
网络上的所有信息都是真实可信的。（ ）

**讲解要点**：
- 网络信息良莠不齐，有真有假
- 要学会辨别
- 答案：×

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________  机位号：________

□ 任务一：搜索引擎使用（10分）
   操作步骤：
   ① 打开浏览器
   ② 地址栏输入：www.baidu.com
   ③ 在搜索框输入：计算机网络

   ⑤ 查看搜索结果，记录前3条结果的标题

□ 任务二：精确搜索（10分）
   操作步骤：
   ① 在百度中搜索："IP地址"（带引号）
   ② 对比不带引号的搜索结果
   ③ 体会精确匹配的作用
   ④ 记录：带引号搜索结果______条

□ 任务三：下载图片（10分）
   操作步骤：
   ① 百度搜索：校园风景
   ② 点击"图片"标签
   ③ 选择一张喜欢的图片
   ④ 右键 → 图片另存为
   ⑤ 保存到"第6周作业"文件夹

□ 任务四：保存网页（5分）
   操作步骤：
   ① 打开任意网页

   ③ 选择保存位置
   ④ 点击保存

【备用任务】完成后可做：
□ 搜索自己的学校名称，查看搜索结果
□ 尝试使用必应搜索引擎（www.bing.com）
```

**教师巡视要点**：
- 指导学生使用准确关键词
