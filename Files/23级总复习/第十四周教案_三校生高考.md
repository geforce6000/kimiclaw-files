# 第十四周教案（第14周）：三校生高考

> **适用对象**：基础薄弱学生（三年级总复习）  
> **周次**：第14周（6月1日—6月7日）  
> **课时安排**：3课时（复习与答疑）  
> **备注**：三校生高考周

---

## 第一课时：考前重点回顾（教室教学）

### 一、信息技术核心概念回顾（15分钟）

```
必背概念：

1. 信息技术定义
   - 获取、处理、传递、利用信息的技术

2. 计算机组成
   - 硬件：CPU、内存、硬盘、输入/输出设备
   - 软件：系统软件、应用软件

3. 网络基础
   - IP地址格式
   - 域名与IP的关系
   - 网络服务功能

4. Word/Excel/PPT基础
   - 常用快捷键
   - 基本操作
```

---

### 二、易错题回顾（20分钟）

回顾前13周常见错误：
- 内存vs硬盘区别
- IP地址格式
- 文件扩展名
- 首行缩进字符数
