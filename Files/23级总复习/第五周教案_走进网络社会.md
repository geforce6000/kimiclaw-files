## 第一课时：网络基础与IP地址（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
计算机网络：将多台计算机连接起来，实现信息共享

网络的功能：
1. 资源共享（文件、打印机）
2. 信息传输（发消息、传文件）
3. 分布式处理（分工合作）

IP地址：计算机在网络中的"门牌号"
格式：XXX.XXX.XXX.XXX（四个0-255的数字）
示例：192.168.1.1

域名：IP地址的"别名"，方便记忆
示例：www.baidu.com（域名）→ 对应某个IP地址
```

**TCP/IP协议速记**：
- TCP：传输控制协议（保证数据可靠传输）
- IP：网际协议（负责寻址和路由）
- 口诀：TCP保证送到，IP负责找路

---

### 二、考点强化记忆（20分钟）

**考点1：网络功能（填空题）**

教师带领学生记忆：
```
网络三大功能：
1. ______共享（资源）
2. ______传输（信息）
3. ______处理（分布式）
```

**考点2：IP地址格式（选择题）**

教师讲解：
- IP地址由4组数字组成，每组0-255
- 用点号隔开：如 192.168.0.1
- 常见内网地址：192.168.x.x

**考点3：域名与IP（判断题）**

教师强调：
- 域名是为了方便记忆IP地址
- 一个域名对应一个IP地址
- www.baidu.com 是域名，不是IP地址

---

### 三、例题讲解（8分钟）

**例题1：填空题**
计算机在网络中的唯一标识是______地址。

**讲解要点**：
- 答案：IP
- 强调：IP地址是计算机在网络中的"身份证"

**例题2：选择题**
下列属于合法IP地址的是（ ）
A. 192.168.1  B. 192.168.1.1  C. 192.168.1.256  D. www.baidu.com

**讲解要点**：
- A选项只有3组数字，不完整
- B选项格式正确，4组0-255的数字
- C选项256超过255，不合法
- D选项是域名，不是IP地址
- 答案：B

**例题3：判断题**
域名和IP地址是一一对应的关系，一个域名只能对应一个IP地址。（ ）

**讲解要点**：
- 基本是一对一，但大型网站可能一个域名对应多个IP（负载均衡）
- 对于基础题，答案为：√（基本对应关系）

## 第二课时：网络服务与应用（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
常见网络服务：

1. WWW服务（万维网）
   - 就是我们常说的"上网看网页"
   - 使用HTTP协议
   - 网址以http://或https://开头

2. 电子邮件（E-mail）
   - 格式：用户名@域名
   - 示例：zhangsan@qq.com
   - @读作"at"，表示"在"

3. 文件传输（FTP）
   - 用于上传和下载文件
   - 例如：从服务器下载软件

4. 域名解析（DNS）
   - 把域名翻译成IP地址
   - 就像"查号台"
```

**网络道德与安全**：
- 不随意泄露个人信息
- 不传播虚假信息
- 遵守网络法律法规

---

### 二、考点强化记忆（20分钟）

**考点1：网络服务类型（选择题）**

教师带领学生识别：
- 浏览网页 → WWW服务
- 收发邮件 → 电子邮件服务
- 下载文件 → 文件传输服务

**考点2：邮箱格式（填空题）**

教师强调格式：
```
邮箱格式：用户名@域名
示例：zhangsan@qq.com
      ↑用户名  ↑域名
```

**考点3：网络安全（判断题）**

教师强调：
- 不随便点击陌生链接
- 不随意下载不明软件
- 保护好自己的账号密码

---

### 三、例题讲解（8分钟）

**例题1：填空题**
电子邮件地址的格式是______@______。

**讲解要点**：
- 答案：用户名、域名
- 示例：zhangsan@qq.com

**例题2：选择题**
我们在浏览器中输入网址访问网站，使用的网络服务是（ ）
A. 电子邮件  B. 万维网(WWW)  C. 文件传输  D. 域名解析

**讲解要点**：
- 浏览网页使用万维网（WWW）服务
- 答案：B

**例题3：选择题**
以下哪个是合法的电子邮箱地址（ ）
A. zhangsan.qq.com  B. zhangsan@qq.com  C. zhangsan@qq  D. @zhangsan.qq.com

**讲解要点**：
- A选项没有@符号
- C选项缺少域名后缀（.com）
- D选项@符号位置错误
- B选项格式正确
- 答案：B

## 第三课时：网络配置与浏览器（上机教学）

1. 查看本机IP地址
2. 使用浏览器访问网站
3. 查看网页源代码（了解HTML概念）

**强调**：清明节放假前最后一次课，巩固前面所学

---

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________  机位号：________

□ 任务一：查看IP地址（10分）
   操作步骤：
   ① 右键点击任务栏网络图标 → 打开"网络和Internet设置"
   ② 点击"更改适配器选项"
   ③ 双击"以太网"（或本地连接）
   ④ 点击"详细信息"
   ⑤ 查看并记录：
      
      IPv4地址：________________
      子网掩码：________________
      默认网关：________________

□ 任务二：浏览器访问网站（10分）
   操作步骤：
   ① 打开浏览器（Edge或Chrome）
   ② 在地址栏输入：www.baidu.com
   ③ 在搜索框输入"清明节"，搜索
   ④ 点击搜索结果中的"清明节-百度百科"
   ⑤ 浏览页面内容

□ 任务三：查看网页源代码（10分）
   操作步骤：
   ① 在打开的百度百科页面
   ② 按键盘 F12 键（或右键 → 查看网页源代码）
   ③ 观察源代码内容（不需要看懂，了解网页是由代码构成的即可）
   ④ 按 F12 关闭代码窗口

□ 任务四：使用ping命令（5分）
   操作步骤：
   ① 开始菜单 → 搜索"cmd" → 打开命令提示符
   ② 输入：ping www.baidu.com
   ③ 查看显示的内容，找到对应的IP地址
   ④ 记录下来：百度网站的IP是________________

【备用任务】完成后可做：
□ 尝试访问其他网站（如www.qq.com）
□ 查看这些网站的IP地址
```

**教师巡视要点**：
- 帮助学生找到查看IP地址的路径
- 指导不会使用浏览器的同学
- 解释ping命令的作用（测试网络连通性）
