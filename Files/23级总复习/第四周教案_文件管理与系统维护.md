## 第一课时：文件与文件夹管理（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
文件命名规则：
┌─────────────────────────────────────┐
│  主文件名.扩展名                      │
│  如：学号姓名.txt                     │
└─────────────────────────────────────┘

命名规则（必背）：
1. 不能用\/:*?"<>| 这些符号
2. 不区分大小写
3. 扩展名表示文件类型

常见扩展名速记：
- .txt  → 文本文件
- .doc/.docx → Word文档
- .xls/.xlsx → Excel表格
- .ppt/.pptx → PPT演示文稿
- .jpg/.png → 图片文件
- .mp3/.mp4 → 音频/视频
- .exe → 可执行程序
```

**文件与文件夹操作**：
- 新建、复制、粘贴、删除、重命名
- 快捷键：Ctrl+C（复制）、Ctrl+V（粘贴）、Ctrl+X（剪切）

---

### 二、考点强化记忆（20分钟）

**考点1：文件命名规则（填空题）**

教师带领学生默写：
```
文件命名不能使用的符号：\ / : * ? " < > |
（口诀：正反斜杠冒号星，问号引号尖竖线）
```

**考点2：扩展名识别（选择题）**

教师带领学生快速记忆：
- .txt → 文本
- .doc → Word
- .xls → Excel
- .ppt → PPT
- .jpg → 图片

**考点3：文件操作快捷键**

教师讲解：
- Ctrl + C = 复制
- Ctrl + V = 粘贴
- Ctrl + X = 剪切
- Ctrl + S = 保存
- Delete = 删除

---

### 三、例题讲解（8分钟）

**例题1：选择题**
下列文件名中，合法的是（ ）
A. file:name.txt  B. file/name.txt  C. filename.txt  D. file*name.txt

**讲解要点**：
- A选项有冒号（:），不合法
- B选项有斜杠（/），不合法
- D选项有星号（*），不合法
- C选项没有非法符号，合法
- 答案：C

**例题2：填空题**
Word文档的扩展名是______，Excel表格的扩展名是______。

**讲解要点**：
- Word文档：.doc 或 .docx
- Excel表格：.xls 或 .xlsx
- 答案：.docx（或.doc）、.xlsx（或.xls）

**例题3：选择题**
将文件从一个文件夹移动到另一个文件夹，应该使用（ ）
A. 复制+粘贴  B. 剪切+粘贴  C. 复制+剪切  D. 直接拖动

**讲解要点**：
- 复制+粘贴：原文件还在，产生副本
- 剪切+粘贴：原文件消失，移动到目标位置
- 题目要求"移动"，所以用剪切+粘贴
- 答案：B

## 第二课时：系统维护与安全（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
用户权限管理：

1. 管理员（Administrator）
   - 拥有最高权限
   - 可以安装软件、修改系统设置
   - 可以创建/删除其他用户

2. 普通用户（Standard User）
   - 只能使用软件，不能安装
   - 不能修改系统重要设置
   - 保护自己的文件

为什么需要权限管理？
- 防止误操作损坏系统
- 保护个人隐私
- 多人共用电脑时互不干扰
```

**系统维护要点**：
- 定期更新系统（打补丁）
- 定期杀毒
- 定期清理垃圾文件
- 重要文件要备份

**备份方式**：
- 本地备份：复制到U盘、移动硬盘
- 云备份：百度网盘、阿里云盘

---

### 二、考点强化记忆（20分钟）

**考点1：用户权限（选择题）**

教师讲解两种用户类型：
- 管理员：什么都能做（最高权限）
- 普通用户：只能使用，不能修改系统

**考点2：系统维护措施（填空题）**

教师带领学生记忆：
1. 系统更新（打补丁）
2. 杀毒查毒
3. 清理垃圾
4. 备份重要文件

**考点3：备份方式（选择题）**

教师讲解：
- 本地备份：U盘、移动硬盘
- 网络备份：云盘（百度网盘等）
- 口诀：重要文件两地存（本地+云端）

---

### 三、例题讲解（8分钟）

**例题1：选择题**
在Windows系统中，拥有最高权限的用户类型是（ ）
A. 普通用户  B. 管理员  C. 来宾用户  D. 游客

**讲解要点**：
- 管理员（Administrator）拥有最高权限
- 普通用户权限受限
- 答案：B

**例题2：填空题**
计算机系统维护的重要措施包括：系统______、杀毒查毒、清理垃圾和______重要文件。

**讲解要点**：
- 第一空：更新（或打补丁）
- 第二空：备份
- 答案：更新、备份

**例题3：选择题**
下列属于云备份方式的是（ ）
A. 复制到U盘  B. 保存到百度网盘  C. 移动到D盘  D. 刻录到光盘

**讲解要点**：
- 云备份是指上传到网络上的存储空间
- U盘、D盘、光盘都是本地存储
- 百度网盘是网络存储（云存储）
- 答案：B

3. 备份方式：本地备份（U盘）+ 云备份（网盘）

**课后记忆任务**：

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________  机位号：________

□ 任务一：创建规范文件夹（10分）
   操作步骤：
   ① 在D盘（或桌面）右键 → 新建 → 文件夹
   ② 命名为"班级_姓名_第4周"（如：计算机1班_张三_第4周）
   ③ 打开该文件夹，再创建三个子文件夹：
      - 文本文件
      - 图片文件
      - 学习资料

□ 任务二：创建文件并操作（15分）
   操作步骤：
   ① 在"文本文件"文件夹中，右键 → 新建 → 文本文档
   ② 命名为"练习.txt"
   ③ 打开文件，输入以下内容：
      班级：______
      姓名：______
      本周学习内容：文件管理
   ④ 保存并关闭
   ⑤ 复制"练习.txt"到"学习资料"文件夹
      （选中文件 → Ctrl+C → 打开目标文件夹 → Ctrl+V）
   ⑥ 将"文本文件"中的"练习.txt"重命名为"我的练习.txt"

□ 任务三：查看用户账户（10分）
   操作步骤：
   ① 开始菜单 → 设置 → 账户
   ② 查看当前登录的账户名
   ③ 查看账户类型（管理员/标准用户）
   ④ 记录下来：
      账户名：________________
      账户类型：________________

【备用任务】完成后可做：
□ 更改桌面背景（个性化设置）
□ 查看系统版本信息
```

**教师巡视要点**：
- 检查文件夹命名是否规范（有无非法符号）
- 帮助学生掌握复制/粘贴快捷键
- 指导找不到"账户"设置的学生

---

### 三、任务单回收与点评（5分钟）

**回收任务单**，快速点评：
- 强调易错点：
  - 文件夹命名不要用特殊符号（如星号、斜杠）
  - 复制和移动的区别（复制产生副本，移动改变位置）
  - 保存文件用Ctrl+S

**布置课后任务**：
- 整理自己的U盘或电脑文件，建立分类文件夹
- 准备下节课默写：非法符号、扩展名、两种用户类型
