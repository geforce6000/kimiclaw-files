# 24级第十二周教案（第12周）：单元小结与综合实践

> **适用对象**：24级学生（新课教学）  
> **周次**：第12周（5月18日—5月24日）  
> **课时安排**：2课时（1节教室教学 + 1节上机教学）  
> **对应教材**：教学设计6.4《初识虚拟现实与增强现实》+ 复习

---

## 第一课时：第六单元小结（教室教学）

### 一、单元内容回顾（20分钟）

**第六单元：数字媒体技术**

**1. 数字媒体基础（第2周）**
```
五要素：文本、图像、音频、视频、动画
格式：JPG（图片）、MP3（音频）、MP4（视频）
工具：美图秀秀
```

**2. 图像处理（第3周）**
```
构图：三分法、中心构图、对角线构图
色彩：冷暖色、调色三参数（亮度、对比度、饱和度）
作品：胸牌制作
```

**3. 演示文稿（第4-5周）**
```
PPT：由幻灯片组成
制作：确定主题→选模板→加内容→设动画→保存
效果：动画（进入、强调、退出）、切换（淡入淡出）
```

**4. VR/AR技术（第6-7周、第10-11周）**
```
VR：虚拟现实，完全虚拟，需头盔，沉浸体验
AR：增强现实，现实+虚拟，用手机，叠加信息
应用：游戏、教育、导航、购物
```

---

### 二、重点强化（15分钟）

**核心概念对比**：

| 技术 | 环境 | 设备 | 特点 |
|-----|------|------|------|
| 美图秀秀 | 图像处理 | 电脑/手机 | 美颜、调色、加文字 |
| PPT | 演示展示 | 电脑 | 图文并茂、动画效果 |
| VR | 完全虚拟 | VR头盔 | 身临其境 |
| AR | 现实+虚拟 | 手机 | 叠加信息 |

**数字媒体制作流程**：
```
1. 确定主题和内容
2. 选择合适工具（美图/PPT/VR/AR）
3. 收集素材（图片、文字）
4. 制作加工
5. 添加效果
6. 保存分享
```


## 第二课时：综合实践——制作数字作品集（上机教学）

- 制作个人数字作品集PPT
- 展示学习成果

**对应关系**：

---

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________ 机位号：________

□ 任务一：规划作品集结构（5分）
   作品主题：我的数字媒体学习之旅
   结构规划：
   - 封面：标题+姓名
   - 第1页：美图秀秀作品（胸牌截图）
   - 第2页：PPT学习心得
   - 第3页：VR体验感受
   - 第4页：AR应用发现
   - 结束页：谢谢观看

□ 任务二：制作封面（5分）
   ① 打开PowerPoint
   ② 选择简洁主题
   ③ 标题：我的数字媒体学习之旅
   ④ 副标题：班级+姓名
   ⑤ 添加装饰图片

□ 任务三：添加内容页（20分）
   
   第1页-美图秀秀：
   ① 插入胸牌图片（或重新制作一张）
   ② 文字说明：学习了构图和色彩
   ③ 使用了：裁剪、调色、加文字功能
   
   第2页-PPT学习：
   ① 文字：学会了制作演示文稿
   ② 要点：添加幻灯片、插入图片、设置动画
   
   第3页-VR体验：
   ① 文字：体验了VR全景视频
   ② 感受：仿佛身临其境
   ③ 插入VR相关图片
   
   第4页-AR发现：
   ① 文字：发现AR应用广泛
   ② 举例：抖音特效、AR导航
   ③ 插入AR相关图片

□ 任务四：添加动画和切换（3分）
   ① 给每页标题添加"飞入"动画
   ② 设置"淡入"切换效果
   ③ 应用到全部

□ 任务五：保存作品（2分）
   ① 文件名：学号_姓名_数字作品集.pptx
   ② 保存到D盘
   ③ 准备展示

【展示环节】：
□ 教师邀请3-5名学生展示作品
□ 学生简单介绍自己的作品集
```

**教师巡视要点**：
- 指导学生规划结构
- 帮助插入图片
- 提醒添加文字说明
- 检查动画设置
