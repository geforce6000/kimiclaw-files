> **适用对象**：24级学生（新课教学）

## 第一课时：认识数字媒体技术（教室教学）

### 一、上节课内容回顾（5分钟）

**提问学生**：
- 什么是信息技术？
- 我们生活中哪些地方用到了信息技术？

**教师总结**：
- 上节课我们了解了信息技术的基本概念
- 今天开始学习新的单元：数字媒体技术

---

### 二、新课导入（5分钟）

**展示案例**：
- 展示抖音短视频、微信表情包、手机拍照等学生熟悉的数字媒体作品
- 提问：这些作品有什么共同点？

**引出课题**：
- 这些都是"数字媒体"
- 今天学习什么是数字媒体技术

---

### 三、内容讲解（25分钟）

#### 1. 什么是数字媒体技术（10分钟）

**定义讲解**：
```
数字媒体技术 = 用计算机处理文字、图片、声音、视频、动画的技术
```

**五要素板书**：
```
数字媒体五大类型：
1. 文本（文字）—— 如：Word文档、TXT
2. 图像（图片）—— 如：照片、表情包
3. 音频（声音）—— 如：音乐、录音
4. 视频（录像）—— 如：抖音、电影
5. 动画（动的画）—— 如：GIF表情包、动画片
```

**举例说明**：
- 发朋友圈 = 文字+图片（可能还有视频）
- 刷抖音 = 视频+音频
- 发微信语音 = 音频

#### 2. 常见文件格式（10分钟）

**格式讲解**：

| 类型 | 常见格式 | 说明 |
|------|---------|------|
| 图片 | JPG、PNG、GIF | JPG最常用，GIF会动 |
| 音频 | MP3 | 音乐、录音 |
| 视频 | MP4 | 抖音、电影 |
| 文档 | DOC、PDF | Word文档 |

**记忆口诀**：
- 图片JPG，音乐MP3，视频MP4

#### 3. 数字媒体技术的特点（5分钟）

**特点讲解**：
```
四大特点：
1. 数字化——用0和1存储
2. 交互性——可以互动（如点击、评论）
3. 集成性——多种媒体合在一起
4. 艺术性——要美观好看
```

---

### 四、简单练习（8分钟）

**课堂练习1**：填空题
```
数字媒体包括：______、______、音频、______、______
答案：文本、图像、视频、动画
```

**课堂练习2**：选择题
```
下列文件中，属于图片格式的是（ ）
A. 歌曲.mp3    B. 照片.jpg    C. 电影.mp4    D. 文档.doc
答案：B
```

**课堂练习3**：判断题
```
数字媒体技术只能处理图片，不能处理视频。（ ）
答案：×（可以处理文字、图片、音频、视频、动画）
```

---

### 五、课堂小结与作业（2分钟）

**本课重点**：

2. 常见格式：JPG、MP3、MP4
3. 四大特点：数字化、交互性、集成性、艺术性

**课后作业**：

## 第二课时：美图秀秀基础操作（上机教学）

### 一、上机任务说明（5分钟）

**本周任务目标**：
- 学会用美图秀秀打开图片
- 学会添加文字
- 学会应用滤镜
- 保存编辑后的图片

**对应关系**：
- 教室学了"图像"是数字媒体的一种
- 上机练习"图像处理"——用美图秀秀处理图片

---

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```
上机操作任务单
姓名：________  机位号：________

□ 任务一：打开软件和图片（5分）
   操作步骤：
   ① 双击桌面"美图秀秀"图标打开软件
   ② 点击"打开"按钮
   ③ 选择一张图片（可用系统示例图或自己准备的图）
   ④ 点击"打开"

□ 任务二：认识界面（3分）
   观察美图秀秀界面：
   - 顶部：美化、美容、饰品、文字、边框等按钮
   - 左侧：各种功能选项
   - 中间：图片显示区
   - 右侧：效果预览区

□ 任务三：添加文字（10分）
   操作步骤：
   ① 点击顶部"文字"按钮
   ② 点击左侧"输入文字"
   ③ 在弹出的框中输入"我的第一张数字作品"
   ④ 选择喜欢的字体和颜色
   ⑤ 拖动文字到图片下方位置

□ 任务四：应用滤镜（10分）
   操作步骤：
   ① 点击顶部"美化"按钮
   ② 在右侧选择一种滤镜效果
      推荐尝试：清新、甜美、复古
   ③ 点击滤镜查看效果
   ④ 如果不满意可以换其他滤镜
   ⑤ 找到最合适的效果

□ 任务五：保存作品（7分）
   操作步骤：
   ① 点击右上角"保存"按钮
   ② 选择保存位置：D盘或桌面
   ③ 文件名格式：学号_姓名_第1课.jpg
      例如：01_张三_第1课.jpg
   ④ 点击"保存"
   ⑤ 确认文件已保存成功

【拓展任务】（完成快的同学可做）：
□ 尝试"裁剪"功能，把图片裁剪成正方形
□ 尝试"边框"功能，给图片加个简单边框
□ 尝试"旋转"功能，调整图片方向
```

**教师巡视要点**：
- 帮助找不到软件的学生
- 指导文字输入和字体选择
- 提醒滤镜不要选太夸张
- 检查文件命名是否规范

---

### 三、作品展示与点评（3分钟）

**邀请2-3名学生展示作品**：
