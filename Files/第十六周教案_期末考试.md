## 重点复习内容

### 一、计算机基础（必考）
1. 信息技术定义：获取、处理、传递、利用
2. 计算机组成：CPU、内存、硬盘、输入/输出设备
3. 存储单位：B、KB、MB、GB（1024进制）
4. 文件命名规则：不能用 \/:*?"<>|

### 二、操作系统（必考）
1. Windows基本操作
2. 文件管理
3. 快捷键：Ctrl+S（保存）

### 三、网络基础（必考）
1. IP地址格式：XXX.XXX.XXX.XXX
2. 域名与IP地址
3. 网络服务：WWW、电子邮件
4. 搜索引擎：百度

### 四、Word（必考）
1. 界面组成
2. 保存快捷键：Ctrl+S
3. 格式设置：首行缩进2字符
4. 表格制作

### 五、Excel（必考）
1. 单元格地址：列+行（如A1）
2. 求和公式：=SUM()
3. 排序：升序/降序

### 六、PPT（必考）
1. 扩展名：.pptx
2. 播放：F5
3. 动画和切换

### 七、多媒体与新技术（必考）
1. 多媒体五要素：文字、图片、音频、视频、动画
2. VR：虚拟现实
3. AR：增强现实
4. AI：人工智能

## 考试技巧

1. **仔细审题**：看清题目要求
2. **先易后难**：先做会的题
3. **注意时间**：合理分配时间
4. **上机题**：随时保存（Ctrl+S）
5. **检查**：做完后检查一遍
