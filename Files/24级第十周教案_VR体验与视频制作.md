### 三、内容讲解（25分钟）

#### 1. VR视频原理（10分钟）

**概念讲解**：
```
VR视频 = 360度全景视频
特点：
- 上下左右都能看到
- 用鼠标或手指拖动视角
- 仿佛身临其境
```

**VR视频类型**：

| 类型 | 特点 | 例子 |
|-----|------|------|
| 全景视频 | 360度看周围 | 风景、演唱会 |
| VR电影 | 沉浸式观影 | VR短片 |
| 互动视频 | 可以选择视角 | 游戏视频 |

#### 2. VR视频制作简介（8分钟）

**制作方式**：
```
1. 专业设备：全景相机（如：Insta360）
2. 手机APP：部分APP支持360度拍摄
3. 电脑软件：用视频软件拼接
```

**简单制作步骤**：
1. 用全景相机拍摄
2. 导入电脑
3. 用软件拼接成360度
4. 导出VR格式

**注意**：专业制作较复杂，今天主要体验

#### 3. VR视频观看方式（7分钟）

**观看方式**：
```
1. 电脑：鼠标拖动视角
2. 手机：手指拖动或重力感应
3. VR头盔：转头即可看不同方向（最沉浸）
```

**推荐平台**：
- B站（搜索"VR全景"）
- 优酷VR
- 爱奇艺VR

### 四、简单练习（8分钟）

**练习1**：填空题
```
VR视频的特点是______度全景，可以用鼠标拖动视角。
答案：360
```

**练习2**：选择题
```
观看VR视频最沉浸的方式是使用（ ）。
A. 电脑显示器  B. 手机屏幕  C. VR头盔  D. 平板电脑
答案：C
```

**练习3**：判断题
```
VR视频和普通视频一样，只能用固定视角观看。（ ）
答案：×（VR视频可以360度拖动视角）
```
