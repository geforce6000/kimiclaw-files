## 第一课时：Word基础操作（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
Word文档基本操作：

1. 创建文档
   - 开始 → Word → 新建空白文档
   - 快捷键：Ctrl+N

2. 保存文档
   - 文件 → 保存（或另存为）
   - 快捷键：Ctrl+S
   - 注意：首次保存要选择位置和文件名

3. 打开文档
   - 文件 → 打开
   - 快捷键：Ctrl+O

4. 关闭文档
   - 文件 → 关闭
   - 或点击窗口右上角 ×

Word界面组成：
- 标题栏（显示文件名）
- 菜单栏/功能区（各种工具）
- 编辑区（打字的地方）
- 状态栏（显示页数、字数）
```

**常用文件格式**：
- .doc/.docx → Word文档
- .pdf → 便携式文档（不可编辑）
- .txt → 纯文本

## 第二课时：文档格式设置（教室教学）

### 一、重难点梳理（15分钟）

**核心概念板书**：
```
Word格式设置三大要素：

1. 文字格式（字体格式）
   - 字体：宋体、黑体、楷体等
   - 字号：五号、小四、四号等
   - 颜色：黑色、红色等
   - 加粗、倾斜、下划线

2. 段落格式
   - 对齐方式：左对齐、居中、右对齐、两端对齐
   - 行距：单倍、1.5倍、2倍
   - 首行缩进：2字符

3. 页面格式
   - 纸张大小：A4、A3等
   - 页边距：上下左右边距

常用操作位置：
- 开始选项卡 → 字体、段落
- 页面布局 → 页面设置
```

**格式设置口诀**：
- 先选中，后设置（先选文字再改格式）
- 标题居中，正文左对齐
- 段首空两格（首行缩进2字符）

---

### 二、考点强化记忆（20分钟）

**考点1：对齐方式（选择题）**

教师带领学生识别：
- 左对齐：文字靠左
- 居中：文字在中间
- 右对齐：文字靠右
- 两端对齐：文字两边都对齐（正文常用）

**考点2：行距（填空题）**

教师讲解：
- 单倍行距：1倍
- 1.5倍行距：1.5倍
- 2倍行距：2倍

**考点3：首行缩进（填空题）**

教师强调：
- 中文段落首行缩进2字符
- 位置：开始 → 段落 → 首行缩进

---

### 三、例题讲解（8分钟）

**例题1：选择题**
在Word中，将标题文字放在页面中间，应该使用（ ）对齐方式。
A. 左对齐  B. 居中  C. 右对齐  D. 两端对齐

**讲解要点**：
- 标题一般居中显示
- 答案：B

**例题2：填空题**
中文文档段落开头应该______缩进______字符。

**讲解要点**：
- 首行缩进2字符
- 答案：首行、2

**例题3：选择题**
正文内容一般采用（ ）行距。
A. 单倍  B. 1.5倍  C. 2倍  D. 固定值

**讲解要点**：
- 正文常用1.5倍行距，阅读舒适
- 答案：B
