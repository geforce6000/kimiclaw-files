### 三、内容讲解（25分钟）

#### 1. 常见恶意攻击形式（12分钟）

**攻击类型**：

| 攻击类型 | 方式 | 后果 | 防范方法 |
|---------|------|------|---------|
| 病毒木马 | 通过文件、U盘传播 | 偷密码、删文件 | 安装杀毒软件 |
| 钓鱼网站 | 假冒正规网站 | 骗账号密码 | 核对网址 |
| 诈骗电话 | 冒充公检法、客服 | 骗钱 | 不轻信、回拨官方 |
| 木马链接 | 短信、微信里的链接 | 手机被控制 | 不点陌生链接 |
| 弱口令攻击 | 破解简单密码 | 账号被盗 | 设置强密码 |

**口诀**：陌生链接不要点，陌生电话不要信

#### 2. 信息安全技术（10分钟）

**防护技术**：

```
1. 身份验证
   - 密码验证
   - 指纹识别
   - 人脸识别

2. 数据加密
   - 把数据变成乱码
   - 只有密钥能解开

3. 防火墙
   - 网络的"门卫"
   - 拦住可疑访问

4. 杀毒软件
   - 查杀病毒木马
   - 定期扫描
```

#### 3. 自我保护措施（3分钟）

**日常习惯**：
- 定期更新系统和软件
- 不下载不明软件
- 重要数据要备份
- 公共场所不用免费WiFi登录账号

**练习1**：填空题
```
防范病毒木马应该安装______软件。

```

**练习2**：选择题
```
收到短信"点击链接领取红包"，应该（ ）。
A. 立即点击  B. 转发给朋友  C. 不点击，可能是诈骗  D. 先点击看看
答案：C
```

**练习3**：判断题
```
在网吧登录个人银行账号是安全的。（ ）
答案：×（公共场所不安全，可能被记录密码）
```

### 二、分任务操作练习（35分钟）

**任务单**（发给学生）：

```

□ 任务一：了解杀毒软件（8分）
   操作步骤：
   ① 打开浏览器
```
